# SpaceWeather

Aplicativo mobile desenvolvido em **React Native** com **Expo SDK 56** e **TypeScript**, focado em monitoramento climático e eventos espaciais em tempo real. O projeto utiliza dados da **NASA** e **OpenWeatherMap**, alinhado à indústria espacial e aos ODS 9, 11 e 13.

## Funcionalidades
- Dashboard com clima atual e eventos espaciais recentes
- Busca por cidade
- Lista de eventos da NASA (DONKI)
- Favoritos com persistência local
- Modo escuro/claro
- Interface responsiva e animada

## Como Executar

1. Clone o repositório
```bash
git clone https://github.com/SEU_USER/spaceweather.git
```
2 Instale as dependencias

3 Execute o projeto
